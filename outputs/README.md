# outputs/

Captured output artifacts from `scripts/` (and other tooling). Handy for sharing a
result without re-running, or for eyeballing a before/after.

| File | Source |
|------|--------|
| `kg-ingest-demo-showcase.txt` | `./venv/bin/python scripts/demo_showcase.py` — the engine reasoning over the real committed KG across six account scenarios. |

Regenerate any file by re-running its source script; these are snapshots, not
inputs to the app.
